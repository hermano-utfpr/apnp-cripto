# Laboratório de Criptografia

**Passo 1**

Faça o download do laboratório:

![](lab_cripto.png)

[lab_cripto.tar.gz](lab_cripto.tar.gz)

**Passo 2**

Abra um terminal e digite:

```
$ cd Downloads
$ ls
$ tar -xvzf lab_cripto.tar.gz
$ ls
$ cd lab_cripto
$ ls
$ sudo py lab.py Seunome
```

Obs.: a senha é abc123.
